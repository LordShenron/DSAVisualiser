ARRAYS

An array is a data structure, which can store a fixed-size collection of elements of the same data type.


STACKS

A stack is a conceptual structure consisting of a set of homogeneous elements and is based on the principle of last in first out (LIFO).


QUEUES

A Queue is a linear structure which follows a particular order in which the operations are performed. The order is First In First Out (FIFO).


LINKED LIST

Linked lists are data structures that hold data in individual objects called nodes. Nodes hold both the data and a reference to the next node in the list.


GRAPHS

A Graph is a non-linear data structure consisting of nodes and edges.
